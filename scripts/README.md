# Scripts

Scripts are local reproducibility helpers.

- `validate-data.sh`: checks JSON syntax for schemas and samples.
- `summarize-benchmarks.py`: summarizes one benchmark run JSON.
- `analyze-paired-benchmark.py`: computes paired quality deltas and a
  stratified bootstrap interval.
- `reproduce-paper.sh`: runs the current smoke checks.
- `build-pdf.sh`: builds `dist/whitepaper.html` and `dist/whitepaper.pdf`.
